SOC Internship

Internship Summary

Project: Building a Security Operations Center (SOC) from the ground up
‍Company: Netcure – provider of 24/7 security monitoring and alerting

Why the project mattered

The company’s alerts were scattered across many tools, There was no single view of what was happening. The project's goal was to pull every log and alert into one place, make them easy to read, and automate as much as possible.

What I did

  • SIEM setup - Deployed and configured Elasticsearch & Logstash. Wrote Ansible playbooks so the stack can be rebuilt in one command.
  • Reporting - Turned Guardian360 CSV scans into interactive HTML + PDF reports (NL/FR/EN) with Python, JS, HTML and DeepL API.
  • Automation
    Setup n8n and Nginx proxy and built n8n workflows for VocalNotify calls. Engineered script to efficiently trim documents from indices.
  • Incident tooling - Installed TheHive with Cassandra, and linked it to existing Nginx proxy/Elasticsearch containers.

Tech in a nutshell

Ansible • Docker • ELK Stack • n8n • Python • Nginx • WSL • Linux command line • Javascript • HTML/CSS • DeepL API

Impact

  • Single dashboard: Security analysts can see all security events in one place.
  • Fast reports: Vulnerability findings now ship in minutes, not hours.
  • Fewer manual tasks: Playbooks and workflows cut repeat work to almost zero.
