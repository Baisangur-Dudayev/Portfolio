Summary
This project is taking place within the framework of the Project hosting and Network & OS Security courses at Thomas More. In our group project, we're taking on the exciting challenge of creating a robust hosting platform tailored for MySQL applications, all managed through an intuitive API interface. This endeavor requires us to dive deep into the complexities of IT infrastructure and technology integration.

Our project involves setting up and fine-tuning essential components like servers, storage systems, switches, fiber switches, and firewalls. Moreover, we're implementing VMware vSphere to establish a cluster environment, ensuring our hosting infrastructure can scale and remain resilient. Security is a top priority, leading us to incorporate pfSense firewall solutions and OpenVPN for fortified network protection.

But our project isn't just about the technical backend. We're also developing a user-friendly website with advanced hosting capabilities utilising container orchestration. This platform will be the gateway for users to configure and deploy their applications seamlessly.
